# Uptimeplaza Homebrew Tap

Homebrew formulae for [Uptimeplaza](https://github.com/Uptimeplaza) projects.

## Usage

```bash
brew tap uptimeplaza/tap
```

Then install any available formula:

```bash
brew install leita
brew install vardrun
```

To update:

```bash
brew update
brew upgrade leita
brew upgrade vardrun
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [leita](Formula/leita.rb) | Domain registration checker via RDAP |
| [vardrun](Formula/vardrun.rb) | Issue tracking CLI |
